# КОТАНДЕР

Мини-стратегия на гексагональной карте: перемещайтесь по соседним клеткам, собирайте ресурсы, улучшайте тайлы и выполняйте условия выбранного режима.

## Что реализовано

- Гекс-карта радиуса `2` (`19` клеток).
- Типы тайлов: `LAVA`, `WASTELAND`, `FIELD`, `FOREST`, `MOUNTAIN`, `VILLAGE`, `CITY`.
- Каждый тайл имеет:
  - число активации (2..12 по фиксированной последовательности),
  - эффект по ресурсам,
  - возможное улучшение (`upgradeTo`).
- Очередь кубиков из 3 значений (текущий + два следующих).
- Перемещение только на соседний тайл.
- Базовый расход еды/здоровья: `-5` за ход.
- Улучшение текущего тайла за `50` монет.

## Режимы игры

Режим выбирается на стартовом экране и сохраняется в `store.selectedGameMode`.

- `STANDARD`: заработать 100 монет.
- `URBANISM`: построить 10 деревень.
- `FEUDALISM`: построить 5 замков.
- `ANKAP`: заработать 250 монет.

Победа рассчитывается по активному режиму: после каждого хода проверяются метрики игрока `FOOD`, `MONEY`, `VILLAGE`, `CITY` и сравниваются с требованиями режима.

## Условия завершения

- Поражение: `health <= 0`.
- Победа: все требуемые параметры выбранного режима достигнуты.

## Интерфейс

- В левом `aside` показывается цель текущего режима и прогресс по требуемым параметрам.
- При `health < 20` счётчик еды начинает мигать.
- При низкой еде показывается предупреждение:
  - `Еда заканчивается! Ищите ее в поле и лесу!`
- Система уведомлений поддерживает критичные сообщения:
  - `showNotification(text, color, duration, isCritical = false)`
  - если `isCritical = true`, уведомление не скрывается автоматически и содержит кнопку `Понятно`.

## Стартовые параметры

- `MAP_RADIUS = 2`
- `INITIAL_HEALTH = 50`
- `INITIAL_GOLD = 50`
- `HEX_SIZE = 55`
- Стоимость улучшения тайла: `50`

На старте первый бросок кубика подбирается под ближайший к игроку тайл с положительным приростом еды.

## Технологии

- `Phaser 3` — карта, интерактивность, анимации.
- `Vue 3` + `Pinia` — UI и состояние.
- `Tailwind CSS` — стили.
- Проект одностраничный, основная логика находится в [index.html](/C:/Users/g.homutov/WebstormProjects/hexgame/index.html).

## Запуск

1. Откройте `index.html` в браузере.
2. Выберите режим на стартовом экране.

Если браузер ограничивает работу с локальными файлами, запустите проект через любой локальный HTTP-сервер из корня репозитория.
